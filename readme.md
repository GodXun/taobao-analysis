# 淘宝用户行为数据分析项目

## 项目简介
使用 Python（pandas）对淘宝用户行为数据进行探索性分析，构建用户转化漏斗，定位"高兴趣、低转化"的异常品类，并提出可落地的业务优化建议。

## 数据来源
- 阿里巴巴淘宝用户行为数据集（UserBehavior）
- 样本量：10万条用户行为记录（pv/fav/cart/buy）
- 时间范围：2017-09-11 至 2017-12-03

## 核心发现
通过分析发现，品类 **4181361** 存在严重转化问题：
- 96次用户兴趣（30次收藏 + 66次加购）
- 仅有 **1次购买**
- 兴趣/购买比高达 **48:1**

## 业务建议
1. 优先验证价格是否高于竞品
2. 检查加购时间是否集中在大促前夕
3. 排查配送费/库存/支付流程问题
